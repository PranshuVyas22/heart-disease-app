# Heart Disease Prediction App

This project predicts the likelihood of heart disease using machine learning.

## Features
- Random Forest model
- Streamlit web interface
- Real-time prediction
- Feature importance visualization

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas, NumPy
